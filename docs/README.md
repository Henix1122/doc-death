# GitHub Pages setup

1. Commit and push this repo to GitHub.
2. In GitHub: Settings > Pages.
3. Source: Deploy from a branch.
4. Branch: main (or master), Folder: /docs.
5. Save. Wait 1-5 minutes for the site to publish.

Your URLs will be:
- Home: https://<github-username>.github.io/<repo-name>/
- Privacy: https://<github-username>.github.io/<repo-name>/privacy.html
- Terms: https://<github-username>.github.io/<repo-name>/terms.html

Update contact emails inside docs/privacy.html and docs/terms.html.
